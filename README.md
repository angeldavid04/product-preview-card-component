## 💪 Mi proceso

### Construido con

- HTML5 semántico
- Propiedades personalizadas de CSS
- Flexbox
- Grid

### Lo que aprendí

Aprendí que puedo utilizar grid para definir facilmente componentes de interfaz que reparten su contenido en más columnas a partir de cierta resolución. También, aprendí una forma de definir tamaños de fuentes dinámicos.

### Desarrollo continuo

Me gustaría investigar más sobre las fuentes responsivas y las distintas formas de que una página web sea adaptativa.
De igual modo, me gustaría aprender más sobre flexbox y grid para saber qué me conviene más utilizar dependiendo de la situación.

### Recursos útiles

- [MDN Web Docs](https://developer.mozilla.org/es/) - Este recurso es muy bueno y me ayuda sobre todo a escoger funciones y características que funcionan en cualquier navegador.
